# Pranetra-ai-waste-management
Pranetra is an early-stage prototype focused on solving urban solid waste management challenges through gamification. The idea aims to transform everyday waste habits into engaging actions and digital incentatives.
## Objective 
To contribute towards United Nations Sustainable Development Goals (SDGs). 
## Concept
The project explores the development of a SaaS-based platform that encourages users to adopt better waste management behaviours through:
- Gamified user experience
- Reward based engagement
- AI driven waste handling concepts
